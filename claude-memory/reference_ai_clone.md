Личная операционная система Андрея Брезгина упакована в `/home/claudeuser/projects/ai-clone/`.

**Читать в начале каждой сессии:**
- `role.md` — кто Андрей, принципы, красные линии
- `feedback/` — правила работы (особенно `CANON.md`, `how-to-work-with-me.md`)
- `memory/MEMORY.md` → актуальные файлы памяти по проектам

**Структура:**
- `identity/` — ценности, миссия, видение, биография
- `thinking/` — ментальные модели, искажения, методы решений
- `action/` — приоритеты, режимы работы, привычки
- `voice/` — стиль, словарь, do/don't для публичных текстов
- `growth/` — учёба, валидация гипотез, энергия
- `mastery/` — чужие фреймворки и шаблоны
- `_input/` — сырые материалы для распаковки
