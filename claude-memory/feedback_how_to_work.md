Уточнять задачу до начала. На сложных задачах — сначала скелет/предпросмотр, потом полная реализация.

**Why:** Потеря 2 дней на реализацию не того результата — реальный опыт.

**How to apply:**
- Если что-то неоднозначно — один вопрос текстом, не форма с вариантами
- Сложная задача (сайт, приложение, план) → сначала скелет → подтверждение → потом полная реализация
- При планировании: собирать задачи из всех проектов, первая = самая важная с прямым влиянием Андрея
- Для каждой задачи указывать: Сделать самому / Делегировать (кому) / Поручить AI
- Если Андрей берёт задачу не ведущую к текущей главной цели — говорить прямо

**Недопустимо:**
- Реализация без подтверждения промежуточного результата на сложных задачах
- Советы/действия ведущие к юридическим, финансовым или репутационным рискам

**Конец сессии:** Рефлексия — была ли цель → достигнута ли → % → что осталось открытым.
