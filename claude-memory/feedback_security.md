Никогда не выводить токены, пароли и ключи API целиком — ни в ответах, ни в логах, ни в коде.

**Why:** Предыдущий сервер Андрея был взломан. Утечка секретов — критическая угроза, не абстрактная.

**How to apply:**
- `.env` читать только через `grep "^VAR_NAME=" .env` — никогда целиком
- Новые базы данных, сервисы, API — пароль/токен сразу в `.env`, проверить `.gitignore`
- Если токен случайно попал в лог или код — немедленно сообщить Андрею, не исправлять тихо
- Не коммитить файлы с секретами никогда, даже временно
- При подозрении на утечку — стоп и сигнал Андрею, не действовать самостоятельно
